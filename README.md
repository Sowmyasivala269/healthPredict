#  Patient Health Risk Prediction
### Masters Research Project — Saint Louis University

## My Contributions (Data Analyst for this Project)
- Exploratory Data Analysis (EDA) on EHR data
- Data cleaning and merging patient, observation 
  and condition tables using Pandas
- Applied SMOTENC to handle class imbalance
- Built and trained XGBoost predictive model 
  achieving ROC-AUC > 0.98
- SHAP visualization to identify top risk 
  contributors per patient
- Model evaluation, validation and interpretation

##  Project Overview
AI-powered health dashboard predicting heart disease 
risk using Electronic Health Record (EHR) data. 
Users can view personalized risk scores, top risk 
factors, preventive suggestions, and book appointments.
